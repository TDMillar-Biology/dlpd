# SVMU2

## Structural Variants from MUMmer Alignments

**SVMU2** is a graph-based structural variant detection and
visualization toolkit for whole-genome alignments generated with MUMmer
(`nucmer`).\
It operates directly on `.delta` files, constructs alignment block
graphs, extracts optimal synteny paths using Dijkstra's algorithm, and
classifies structural variants using geometric relationships between
colinear segments.

## Abstract

Structural variant (SV) detection from whole-genome alignments requires
distinguishing true rearrangements from fragmented or repeat-mediated
alignment artifacts.\
SVMU2 implements a graph-theoretic framework to model alignment blocks
as nodes in a directed graph, with weighted transitions encoding
colinearity constraints. Optimal syntenic paths are extracted via
Dijkstra's algorithm, and structural variants are inferred from
discontinuities in these paths using vector-based geometric
classification.

SVMU2 is designed for research use in comparative genomics, assembly
validation, and structural variation analysis.

------------------------------------------------------------------------

## Features

-   Structural variant detection from MUMmer `.delta` files
-   Detection of:
    -   Deletions (DEL)
    -   Insertions (INS)
    -   Tandem duplications (DUP_TANDEM)
    -   Inversions (INV)
    -   Breakends (BND; optional)
    -   Complex events (COMPLEX)
-   Static dotplot visualization (PNG)
-   Interactive dotplots (HTML)
-   Graph-based synteny modeling
-   Experimental scaffolding module

``` bash
cd svmu2
```

Install in editable mode:

``` bash
pip install -e .
```

-   pysam
-   matplotlib
-   plotly
-   networkx
-   intervaltree

## Input Requirements

SVMU2 requires a MUMmer `.delta` file generated by `nucmer`:

``` bash
nucmer -p sample reference.fasta query.fasta
```

This produces:

    sample.delta

------------------------------------------------------------------------

## Command-Line Interface

SVMU2 uses subcommands:

    svmu2 <command> [options]

Available commands:

-   `plot` -- Static dotplot visualization
-   `interactive` -- Interactive dotplot (HTML)
-   `call` -- Structural variant detection (VCF output)
-   `scaffold` -- Experimental scaffolding
-   `trend` -- Debugging visualization

------------------------------------------------------------------------

## Structural Variant Calling

``` bash
svmu2 call     --delta sample.delta     --out sample.sv.vcf     --sample SAMPLE_NAME
```

Optional:

    --write-bnds   # Output explicit breakend records

------------------------------------------------------------------------

## Static Dotplot

``` bash
svmu2 plot     --delta sample.delta     --out_dir plots/
```

Optional filters:

    --contig NAME
    --reftarget NAME
    --qrytarget NAME
    --all
    --popup
    --strain NAME

## Interactive Dotplot

``` bash
svmu2 interactive     --delta sample.delta     --out_dir html_plots/
```

## Scaffolding (Experimental)

``` bash
svmu2 scaffold     --delta sample.delta     --reference reference.fasta     --query query.fasta
```

## Methodological Framework

SVMU2 operates in five conceptual stages:

1.  Parse alignment blocks from MUMmer `.delta` files.
2.  Construct a directed graph where nodes represent alignment segments.
3.  Weight edges according to colinearity and distance constraints.
4.  Extract optimal synteny paths via Dijkstra's algorithm.
5.  Infer structural variants from discontinuities and geometric
    deviations.

This framework allows robust discrimination between true structural
rearrangements and fragmented alignment artifacts.

------------------------------------------------------------------------

## Intended Use

SVMU2 is intended for:

-   Comparative genome analysis
-   Assembly validation
-   Structural variation research
-   Methodological development in graph-based genomics

It is not currently optimized for population-scale short-read SV
detection.
